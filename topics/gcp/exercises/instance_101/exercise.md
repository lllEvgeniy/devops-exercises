# Создание экземпляра VM

## Цели

1. Создайте виртуальную машину со следующими свойствами:
   1. имя: `instance-1`
   2. тип машины: `e2-micro`
   3. **labels**:
      1. `app=web`
      2. `env=dev`
2. С помощью **gcloud** выполните:
   1. Измените метку `app` на `db`.
   2. Удалите метку `env`.
